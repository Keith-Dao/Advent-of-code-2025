# Advent-of-code-2025

## Setup

**Using UV:**

1. Run `uv sync` if dev dependencies are not required run `uv sync --no-dev`

**Not using UV:**

1. Create and activate a virtual environment (Optional)
2. Install using `pip install .` if dev dependencies are required run `pip install .[dev]`

## Usage

Run `solver <DAY> <FILEPATH>`

- \<DAY> - Integer in [1, 12]
- \<FILEPATH> (optional) - Input filepath. If not provided, defaults to `input.txt`
